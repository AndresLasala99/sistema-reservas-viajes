# Sistema de Reservas de Viajes

Proyecto académico desarrollado para la materia Programación 2.

## Descripción
Aplicación desarrollada en C# y .NET que permite gestionar reservas de viajes,
usuarios y la lógica de negocio separada por capas.

## Estructura del proyecto
- Aplicación Web: interfaz de usuario
- Dominio: lógica de negocio y entidades
- Obligatorio.sln: solución principal del proyecto

## Funcionalidades
- Gestión de usuarios
- Registro y administración de reservas
- Validaciones de negocio
- Separación por capas

## Tecnologías utilizadas
- C#
- .NET
- Visual Studio

Proyecto realizado con fines académicos.
